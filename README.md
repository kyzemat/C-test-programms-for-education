# Изучение C++ по книге Герберта Шилдта

Этот репозиторий содержит мой код, написанный в процессе изучения книги 
**"C++: Базовый курс"** (Herbert Schildt).

## Цель
- Систематизировать практические примеры из книги
- Продемонстрировать понимание ключевых концепций C++
- Создать портфолио учебных проектов для будущих работодателей

## Структура
Код организован по главам/темам. Каждый пример сопровождается комментариями 
с пояснениями ключевых моментов.

## Как использовать
1. Клонируйте репозиторий
2. Открывайте нужный файл в вашей IDE
3. Компилируйте и запускайте

## Дополнительно
Основные конспекты веду в Obsidian, здесь публикую только рабочий код.
